## Overview (TL;DR)

A Lenovo LOQ Full HD Gaming Laptop was upgraded from 12 GB to 32 GB of DDR5 RAM to improve multitasking performance and system responsiveness. After verifying compatibility and performing the installation, the system demonstrated smoother multitasking and improved performance under load.

---

## Background / Context

The user reported:

- System slowdowns during multitasking  
- Performance drops when running multiple applications  
- Occasional lag during gaming and heavy workloads  

---

## Problem Statement

The installed 12 GB of RAM was insufficient for the user’s workload, leading to memory pressure and reduced performance during multitasking.

---

## Environment & Tools

### Device Information

- **Device:** Lenovo LOQ Full HD Gaming Laptop  
- **OS:** Windows 11  

### Hardware Used

- Crucial 32GB DDR5 RAM Kit (2×16GB)

### Tools Used

- Phillips #00 Screwdriver  
- Phillips #000 Screwdriver  
- Flashlight  
- Tweezers  
- Triangle Prying Picks  

---

## Troubleshooting Process

1. Verified user-reported performance issues during multitasking.  
2. Checked memory utilization in Task Manager.  
3. Observed high memory usage under normal workload conditions.  
4. Confirmed laptop RAM compatibility and maximum supported capacity.  
5. Powered down device and safely opened chassis.  
6. Removed existing memory modules.  
7. Installed Crucial 32 GB DDR5 kit.  
8. Reassembled device and powered on system.  
9. Verified full memory recognition in BIOS and Windows.

---

## Root Cause

The original 12 GB RAM configuration was insufficient for the user’s multitasking and performance requirements, causing memory bottlenecks.

---

## Resolution

- Upgraded system memory from **12 GB → 32 GB DDR5**  
- Confirmed proper seating and dual-channel operation  
- Validated memory recognition in Windows 11  

---

## Results / Impact

- Improved multitasking performance  
- Reduced system lag under heavy workloads  
- Smoother gaming and application performance  
- Increased system headroom for future workloads  

---

## Lessons Learned

- Proper RAM capacity planning is critical for performance-heavy users.  
- Verifying system compatibility prevents post-installation issues.  
- Post-upgrade validation in both BIOS and OS is essential.
